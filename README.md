# Symposium Participation Proposal

## Building a Secure AI Knowledge Assistant with Retrieval-Augmented Generation (RAG)

### Abstract / Presentation Description (≤150 words)

How can institutions safely use generative AI with their own internal data? This session presents a practical project exploring Retrieval-Augmented Generation (RAG) using UCSB's LLM Sandbox and support ticket data from Zoho Desk. Instead of sending sensitive information to external AI services, we demonstrate how institutional tools can securely retrieve relevant content and provide natural-language answers grounded in trusted sources.

The project focuses on building a simple chat interface that allows users to query internal knowledge—such as support tickets, documentation, or common issues—using conversational prompts. Behind the scenes, the system retrieves relevant records and uses the LLM Sandbox API to generate responses while keeping data within university-controlled infrastructure.

We will share lessons from the prototype, including data preparation, token and cost considerations, and practical implementation challenges. The session highlights how small teams can build responsible, secure AI tools that improve knowledge access while aligning with institutional privacy and governance standards.
